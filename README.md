## How to Use
```xml
<repository>
  <url>http://shiruka.minecraftevi.com:8081/repository/maven-public/</url>
  <id>shiruka</id>
</repository>

<dependency>
    <groupId>io.github.shiruka</groupId>
    <artifactId>api</artifactId>
    <version>${version}</version>
</dependency>
```
```groovy
implementation("io.github.shiruka:api:${version}")
```
